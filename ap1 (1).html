Ateliers 1ère année

Situations professionnelles réalisées en première année de BTS SIO SISR : installation de GLPI et mise en place d'un serveur Web.

Atelier 1

Installation de GLPI

Contexte

Dans cet atelier, nous avons appris à installer et utiliser GLPI sur un serveur GNU/Linux (Ubuntu), afin de nous initier aux systèmes de ticketing très utilisés en entreprise.

Objectifs

Chaque entreprise dispose d'un parc informatique nécessitant une gestion à distance (inventaire, tâches administratives) et une plateforme d'assistance fonctionnant sur la base de signalement d'incidents. L'objectif est de mettre en place GLPI, logiciel libre de gestion de parc informatique.

Environnement technologique

Ubuntu 16 (VM)
Apache (serveur Web)
MySQL
PHP
GLPI

Réalisation

  • Mise à jour des paquets de la machine
  • Installation des services complémentaires (Apache, MySQL, PHP)
  • Installation de GLPI
  • Mise en service via navigateur Web : création utilisateur et parc informatique
  • Gestion des tickets d'incidents

Compétences mobilisées

  • Gérer le patrimoine informatique : recenser les ressources, mettre en place les niveaux d'habilitation
  • Mettre à disposition un service : tests d'intégration, déploiement, accompagnement utilisateurs
  • Répondre aux incidents : collecter et orienter les demandes, traiter les demandes réseau et applicatives

Atelier 2

Installation d'un serveur Web

Contexte

Dans le cadre des ateliers Support & Système de 1ère année BTS SIO SISR : installation et mise en service d'un serveur HTTP à destination d'une entreprise.

Objectifs

Une entreprise disposant d'un service en ligne dépend d'un serveur HTTP permettant l'offre de service et la mise à jour des informations. Ce serveur doit être accessible à distance pour que les collaborateurs puissent le modifier à tout moment.

Environnement technologique

Debian Server 10 (VM serveur)
Debian Client 10 (VM client)
Apache
MariaDB + phpMyAdmin
PHP
ProFTPD

Réalisation

Installation et configuration d'un serveur HTTP permettant à plusieurs collaborateurs d'accéder aux applications de manière fiable et sécurisée. Le serveur FTP ProFTPD assure la gestion à distance des fichiers disponibles sur le serveur Web.

Compétences mobilisées

  • Développer la présence en ligne de l'organisation : participer à l'évolution d'un site Web
  • Mettre à disposition un service : tests d'intégration et déploiement
